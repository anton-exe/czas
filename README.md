# Czas
short and bad game for a uni project

## Building
to build, install Maven and in this repo's root run:
```sh
mvn clean package
```
the jar file will be located at `target/Czas.jar`

## Running
either double-click in your favourite file manager or
```sh
java -jar Czas.jar
```

## Usage
in-game, type `help` in the input line and press enter for a list of commands
